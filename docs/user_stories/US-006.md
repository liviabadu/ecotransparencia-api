# US-006: Visualizar detalhes do embargo

## Descrição

**Como** um usuário do sistema
**Quero** ver os detalhes completos de cada embargo
**Para** entender a natureza e gravidade da infração ambiental

## Contexto

Esta funcionalidade enriquece os dados retornados em cada ocorrência (embargo), adicionando informações detalhadas sobre localização geográfica, bioma afetado, área embargada e dados do auto de infração. Esses detalhes são essenciais para análises aprofundadas de compliance ambiental.

## Endpoints Afetados

Todos os endpoints de busca que retornam ocorrências:
```
GET /api/search/document?document={cnpj}&type=cnpj
GET /api/search/document?document={cpf}&type=cpf
GET /api/search/name?name={termo}
```

## Critérios de Aceitação

### AC-01: Exibir campos básicos do termo de embargo
- **Dado** que uma ocorrência é retornada
- **Então** deve incluir os campos básicos:
  - `id`: Identificador único (SEQ_TAD)
  - `category`: Tipo de área embargada (TP_AREA_EMBARGADA)
  - `date`: Data do embargo (DAT_EMBARGO) em formato ISO 8601
  - `description`: Descrição/justificativa (DES_TAD)
  - `source`: "IBAMA" (fixo)
  - `sourceUrl`: URL para consulta construída
  - `status`: "Ativo" ou "Baixado"

**Teste:** Validação dos campos básicos existentes

### AC-02: Incluir localização geográfica
- **Dado** que o embargo possui dados de localização
- **Então** deve incluir o objeto `location` com:
  - `uf`: Sigla da UF (SIG_UF_TAD)
  - `municipio`: Nome do município (NOM_MUNICIPIO_TAD)
  - `imovel`: Nome do imóvel (NOME_IMOVEL)

**Teste:** Validação do objeto location na resposta

### AC-03: Incluir informações do auto de infração
- **Dado** que o embargo possui auto de infração vinculado
- **Então** deve incluir o campo `autoInfracao` com número e série

**Teste:** Validação do campo autoInfracao

### AC-04: Indicar o bioma afetado
- **Dado** que o embargo possui informação de bioma
- **Então** deve incluir o campo `biome` com o nome do bioma (DES_TIPO_BIOMA)

**Teste:** Validação do campo biome na resposta

### AC-05: Indicar se é desmatamento
- **Dado** que o embargo possui indicador de desmatamento (SIT_DESMATAMENTO)
- **Quando** o valor é "D"
- **Então** o campo `desmatamento` deve ser `true`
- **Quando** o valor é "N" ou nulo
- **Então** o campo `desmatamento` deve ser `false`

**Teste:** Validação do campo desmatamento

### AC-06: Incluir área embargada
- **Dado** que o embargo possui área embargada (QTD_AREA_EMBARGADA)
- **Então** deve incluir o campo `areaEmbargada` com valor em hectares

**Teste:** Validação do campo areaEmbargada

## Regras de Negócio

### RN-01: Campo autoInfracao
- Concatenar `NUM_AUTO_INFRACAO` com `SER_AUTO_INFRACAO` usando hífen
- Exemplo: "789209" + "A" = "789209-A"
- Se série for vazia, retornar apenas o número

### RN-02: Campo desmatamento
- Converter `SIT_DESMATAMENTO` para boolean:
  - "D" → `true` (desmatamento/queimada)
  - "N" ou null → `false` (outras situações)

### RN-03: Campo biome
- Usar o valor de `DES_TIPO_BIOMA` diretamente
- Valores possíveis: "Amazônia", "Mata Atlântica", "Cerrado", "Caatinga", "Pampa", "Costeiro e Marinho"
- Pode ser null se não informado

### RN-04: Objeto location
- Incluir apenas se houver dados de localização
- Campos podem ser null individualmente
- Não incluir se todos os campos forem null

### RN-05: Campo areaEmbargada
- Usar o valor de `QTD_AREA_EMBARGADA` em hectares
- Pode ser null se não informado
- Não serializar se for null

### SearchService.java (Atualizado)

```java
private OccurrenceDto toOccurrenceDto(Embargo embargo) {
    OccurrenceDto dto = new OccurrenceDto();
    // ... campos existentes ...

    // US-006: Campos adicionais
    dto.setAutoInfracao(formatAutoInfracao(embargo));
    dto.setDesmatamento("D".equals(embargo.getSitDesmatamento()));
    dto.setAreaEmbargada(embargo.getQtdAreaEmbargada());
    dto.setBiome(embargo.getDesTipoBioma());
    dto.setLocation(buildLocation(embargo));

    return dto;
}

private String formatAutoInfracao(Embargo embargo) {
    String num = embargo.getNumAutoInfracao();
    String ser = embargo.getSerAutoInfracao();
    if (num == null) return null;
    if (ser == null || ser.isEmpty()) return num;
    return num + "-" + ser;
}

private LocationDto buildLocation(Embargo embargo) {
    if (embargo.getSigUfTad() == null &&
        embargo.getNomMunicipioTad() == null &&
        embargo.getNomeImovel() == null) {
        return null;
    }
    LocationDto location = new LocationDto();
    location.setUf(embargo.getSigUfTad());
    location.setMunicipio(embargo.getNomMunicipioTad());
    location.setImovel(embargo.getNomeImovel());
    return location;
}
```
